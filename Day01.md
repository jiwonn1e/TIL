# **TIL day 01**
터미널 켤 때 ctrl+` (tab위에 키)
1. 각각 로컬 만들기
폴더만들어서 파일하나 생성
git init

2. 연결 
git remote add origin 원격저장소주소
git remote -v

3. 업로드
1. git add 파일명
2. git commit -m "이유"
(보기 : git log)
3. git push origin master
100% 나오면 완료 된것
